# 그래프 전시 + FFT

## 목적
바이너리 파일을 읽어 **시간영역 그래프**와 **FFT 스펙트럼**을 보여주는 가이드입니다.

## 구현 포인트
- `matplotlib` + `numpy` 사용
- 파일 로드 후 `uint8` 배열로 변환
- 구간 선택(시작/끝) 후 그래프 출력
- 다운샘플링으로 렌더링 부담 감소
- FFT는 Hanning 윈도우 적용 후 `rfft`로 계산

## 확대/축소
`matplotlib` 기본 툴바(돋보기/드래그)를 통해 확대/축소 가능
